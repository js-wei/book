  • 标题(title)是在浏览器上面显示出来的,方便用户了解这个页面的内容,特别是搜索引擎判断你网页内容的主要根据
  • 网页标题不能太长,一般不超过80个字符
  • 首页的title写法格式一般是"总标题-特别重要的关键词或者一句话含有特别重要关键词的描述"
  • 网站页title,网站页title的写法有两种,关键词名称"网站名称-总名称",非关键词命名:"网站名称 网站关键词-总名称"
  • 分类列表页的title写法:用关键词为这个网站起名,然后按照下列顺序填写便可了"分类列表页名称-网站名称-总名称"
  • 内容的title写法:用关键词为这个文档起名,然后按照下列顺序填写便可了"文档关键词-分类列表页名称-网站名称-总名称"
  • 标题是搜索引擎判断你网页内容的主要依据应重视
  • 关键词(keywords)访问者不可见的,通过查看源代码可以看到,主要作用是告诉搜索引擎,这个网站内容是什么
  • 网页关键词不能太长,一般不超过100个字符
  • 首页keywords按照选定的网站名称,在首页的keywords中加入总名称、网站名称和一两个关键词
  • 网站的keywords其网站下所有分类列表的名称列出,加上网站关键字,写法是"网站名称,网站关键字,网站分类列表名称"
  • 分类列表页keywords写法,将你这个网站中的主要关键字写入
  • keywords要本着为全站服务,关键词要体现全站的主题
  • 描述的滥用,导致目前常用的搜索引擎降低了keywords的重要性已降权
  • 描述(description),提供的网页简短描述通常是为搜索引擎描述网页使用的
  • 网页描述不能太长,应该保持在140-200个字符或者100个左右的汉字
  • 不要给网页定义与网页描述内容无关的简短描述
  • 描述的滥用,导致目前常用的搜索引擎降低了description的重要性已降权
